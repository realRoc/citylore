# Contributing

This repository accepts two contribution modes:

1. Direct pull requests to improve shared data
2. Forks that keep their own local context and can be federated later

For the MVP:

- Add canonical place or area data under `data/`
- Add personal taste under `data/profiles/`
- Add personal opinions under `data/opinions/`
- Put imported external material under `imports/`, not under `data/`
- Add cross-entity linkage under `relations/`

Keep text files structured and small so both humans and agents can inspect them easily.
